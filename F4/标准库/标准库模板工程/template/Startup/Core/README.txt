启动的汇编文件(*.s)根据具体型号的芯片选择(该模板使用stm32f427所以选startup_stm32f427_437xx.s)

 - core_cmSimd.h: 
	用于处理器在同一时间内对多个数据进行相同的操作，从而提高处理向量数据或执行并行计算的效率，特别是在信号处理和数学运算等应用中。在使用Cortex-M4或M7的项目中.

- core_cmFunc.h: 
	通过包含core_cmFunc.h，开发者可以方便地进行以下操作：
	处理器状态管理：例如，切换处理器的工作模式，如从线程模式到中断模式。
	中断处理：设置和清除中断优先级，进入和退出中断服务例程。
	处理器控制：访问和修改像NVIC（Nested Vectored Interrupt Controller）这样的系统控制器寄存器，用于管理中断和异常。
	其他低级别控制：如设置处理器的堆栈指针，或进行其他与处理器架构相关的操作。

core_cmInstr.h 
	文件同样是CMSIS库的一部分，它是专门为ARM Cortex-M系列微控制器设计的。
	这个头文件包含了针对Cortex-M处理器的指令集定义和宏，这些宏通常用于直接操作处理器的指令，比如读写特定的CPU寄存器或者执行特定的汇编指令。
